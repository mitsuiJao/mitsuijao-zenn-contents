## ポートフォワーディングを使う

もちろんSSHで踏み台にできるサーバにアクセスできることが必要です

```
ssh -L 8888:{homerouterIP}:80 {host}@{user} -N -v -g
```

ホームルータのIPを指定します。
そしてブラウザで`localhost:8888`でいけます

外からコンフィグ変えたいときありますよね。そんな時に使います
