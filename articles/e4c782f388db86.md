### 代数

$$
E = (x^2-4)(x+2)
$$

```python
E = (x**2 - 4) * (x + 2)        # 定義

expand(E)                       # 展開
factor(E)                       # 因数分解
simplify(E)                     # 簡易化
E.subs(x, 3)                    # x=3 代入

equation = Eq(E, 0)             # 方程式の定義、左辺=右辺
solve(equation, x)              # xの解
```

### 微積分

$$
f(x) = x^3 + 4x^2 - 4x - 16
$$

```python
f_x = x**3 + 4*x**2 - 4*x - 16  # 定義

diff(f_x, x)                    # 微分
integrate(f_x, x)               # 不定積分
integrate(f_x, (x, 0, 1))       # 定積分    0->1
```

$$
f(x) = \frac{\ln(x)}{x}
$$

```python
f_x = log(x) / x                # 定義

limit(f_x, x, oo)               # 極限     x->無限
```
